# Perggo Public Prototype (Phase 3)

Static public prototype served from `frontend/public/perggo-prototype/`.

## Scope

- Plain HTML/CSS only.
- No React app integration.
- No frontend/src or backend changes.
- No binary/image files included in this folder.

## Deploy path

After Vercel deploy, access at:

- `/perggo-prototype/`

## Image preview policy (GitHub raw)

This public prototype renders visual previews by referencing **remote GitHub raw URLs** for existing repository assets.

- Logo previews use files from `frontend/src/assets/perggo/`.
- Screen previews use files from `docs/perggo-design/reference-screens/`.
- No PNG/image files are copied into `frontend/public/perggo-prototype/`.

If the repository is private, or if GitHub raw access is blocked by network/browser policy, image previews may fail to load in this page.

The source of truth remains:

- `docs/perggo-design/reference-screens/`
- `frontend/src/assets/perggo/`
