# Perggo Prototype (Phase 2)

## Purpose

This folder contains **Phase 2** of Perggo “Figma in Code”: a **static HTML/CSS visual prototype/documentation page**.

It is documentation-only and does not change app behavior.

## Files

- `index.html` — static prototype page with Perggo visual sections.
- `styles.css` — plain CSS for layout, responsive behavior and visual language.
- `README.md` — this documentation.

## Scope and constraints

- Static documentation/prototype only.
- No frontend app code changes.
- No backend code changes.
- No runtime behavior changes.
- No auth, tenant, company, marketplace, notification or AI logic changes.
- No build tooling.
- No external dependencies.
- Plain HTML and CSS only.

## Asset linking policy

This prototype references existing assets using relative paths only.

Logo assets:

- `../../../frontend/src/assets/perggo/perggo-icon-source.png`
- `../../../frontend/src/assets/perggo/perggo-logo-source.png`
- `../../../frontend/src/assets/perggo/perggo-wordmark-source.png`

Reference screens:

- `../reference-screens/*.png` (all 27 official screen images)

No image assets are copied, moved, renamed, duplicated or modified.

## Local preview

Open directly in a browser:
